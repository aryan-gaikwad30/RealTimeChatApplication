# 💬 RealTimeChatApp

A modern real-time chat application built with **Flask**, **Flask-SocketIO**, **SQLite**, **HTML**, **CSS**, and **JavaScript**, featuring one-to-one messaging, group chats, typing indicators, GIF support, emoji integration, and a responsive user interface.

## ✨ Features

- 🔐 User Authentication
- 💬 One-to-One Chat
- 👥 Group Chat
- ➕ Create Groups
- 👤 Add Members
- 🚪 Leave Group
- ⌨️ Typing Indicator
- 🟢 Online / Offline Status
- 🕒 Last Seen
- 😀 Emoji Picker
- 🎞️ GIF Search (Giphy API)
- 📜 Auto Scroll
- 🕓 Message Timestamps
- 👥 Group Member Count
- 📱 Responsive UI
